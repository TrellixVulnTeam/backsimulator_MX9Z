## Requirements

* [Python 2.7]

### Installation

Clone repository and install depedencies.

```sh
$ git clone https://github.com/MSLacerda/BackEco
$ cd BackEco
$ pip install pip install -r requirements.txt
```
### Running
```sh
$ python wsgi.py
```

### Commits

Commit should be maked in **dev** branch, and must be created a **pull request**.
