## Requirements

* [Python 2.7]

### Installation

Clone repository and install depedencies.

```sh
$ git clone https://github.com/joaowillamy/world-cup-20018.git
$ cd world-cup-20018
$ sudo pip install -r requirements.txt
```
### Running
```sh
$ python wsgi.py
```
